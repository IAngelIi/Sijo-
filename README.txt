SIJO clan website (static)
Files:
- index.html
- about.html
- achievements.html
- members.html
- css/style.css
- js/main.js

How to use:
Open index.html in a browser. This is a static front-end only project. Edit members in js/main.js (the `members` array) to update roles and descriptions.

Role gradients implemented:
- Owner: animated grey→white (text gradient)
- Co-Owner: #4b6cb7 → #182848
- Elder: #dc2424 → #4a569d
- Member: #603813 → #b29f94
